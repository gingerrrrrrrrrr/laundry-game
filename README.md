## 🎮 游戏玩法

游戏采用三列布局：
- **左列**：待处理订单（顾客送来的衣服）
- **中列**：洗衣机（洗涤衣物）
- **右列**：晾晒架（晾干衣物）

核心循环：接单 → 洗涤 → 晾晒 → 收钱

## ✨ 特色功能

- ✅ 双光标自动化系统（👆 和 🖱️ 自动找任务、移动、点击）
- ✅ 预定表机制防止光标抢单
- ✅ 玩家点击可打断光标任务
- ✅ 洗衣机/晾晒架购买与升级
- ✅ 加急订单系统

## 🚀 如何运行

1. 克隆本项目
2. 安装依赖：`npm install`
3. 启动项目：`npm start`
